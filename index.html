CSS Selectors & Specificity

CSS Selectors

Selector Description Specificity
* {} Applies CSS styles to all elements in a document unless otherwise specified. Least specific. Can be overridden by element, class or id selectors.
element {} Applies styles to all elements with the selected tag. Second least specific, can be overridden by id or class selectors.
.classname {} Applies styles to all elements with the selected class name More specific than element selectors. Can be overridden by id selectors.
#id {} Selects an element with corresponding ID and applies styles Second most specific. Overrides class, element and universal styles.

Uses Specificity without a Selector

Attribute Description Specificity
Inline Styles Used to write CSS code in a html document. Applies styles to the element in which it is written. Usually most specific. Overrides all selectors unless '!important' is used.
!important Overrides all other CSS rules. Should only be used if a more specific selector cannot be used. Most specific. Forces the selected CSS rule to always apply regardless of specificity.
